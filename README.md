# RTL Design Portfolio

This repository contains a collection of synthesizable SystemVerilog RTL
design projects created as part of focused preparation for RTL design
engineering roles.

The goal of this portfolio is to demonstrate practical experience in
digital hardware design, including datapath construction, control logic,
interface design, and simulation-based verification.

Each project includes:
- synthesizable SystemVerilog RTL
- simulation testbenches
- waveform verification
- design documentation and architectural notes
